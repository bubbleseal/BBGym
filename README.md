# BBGym

## BB Gymnastics Centre (2015)
[BB Gymnastics Centre](https://bbgimn-51605.firebaseapp.com/) is a website developed from scratch with HTML, CSS and Javascript. 
The client: BetterBounce Gymnastic Centre (BB Gymnastic Centre) have been conducting gymnastics classes for children in kindergartens and schools, and have been doing so since 1980s.
The website is currently undergoing upgrades to improve the UI, perfomance and sustainability.
